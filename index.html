Projects

Hero, Villain, and Victim : Semantic role labelling of entities

The project is done as part of and dataset provided by https://constraint-lcs2.github.io/. We aim to project a particular entity as either hero, villain or victim based on the context it is mentioned in. The project deals with multimodal analysis, alignment of entity with visual and textual context. We test out models using different alignment technique, different multimodal fusion and compare the results. [Github] [Report]

GAN based Video Colorization

Leveraging GAN architecture to color grayscale videos. We test out a combination of models utilizing pretrained resnet, encoder-decoder such as U-Net, PATCHGan discriminator, NOGan training, Wasserstein loss, and Perceptual Loss [Github] [Report]

Malaria detection from image cells

We studied in detail the task of detecting malaria given the image of cell. Here, we compare multiple traditional machine learning techniques with deep learning architecture. The aim is to see if its possible to achieve results comparable to deep learning architecture with a combination of image preprocessing such as edge detection, local and global feature extraction and traditional machine learning.[Github] [Report]

Artificial Neural Network

This project is a detailed scratch implemntation of neural network with keeping processing efficiency in mind. Implemented multiple activation function with varying batch sizes option along with Gradient Descent with Momentum, NAG, RMSProp, AdaGrad, and Adam as optimisation techniques.[Github]
